# Chiriqui Eats

Aplicación Android nativa en Java para gestionar empresas, menú y pedidos.

## Abrir en Android Studio

1. Abre Android Studio.
2. Selecciona `Open`.
3. Elige esta carpeta del proyecto.
4. Espera la sincronización de Gradle.
5. Ejecuta la app en un emulador o dispositivo Android.

La pantalla principal usa `BottomNavigationView` con las pestañas `Empresas`, `Menú` y `Pedidos`.
